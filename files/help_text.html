Tox21 data visualisation for EU-ToxRisk

Note that Efficacy data has been trimmed to a maximum absolute value of 200%, as very large values can make scatterplots more difficult to interpret.

To see the data for different assays, use the 'Assay Name' category browser widget to step through them; this is located at the top of the right-hand panel.

To view only data for one or any of the EU-ToxRisk case study compounds, use the set of 'Case Study' check boxes below the Assay Name tool.

To highlight the EU-ToxRisk case study compounds while keeping other compounds visible (for the context they provide)...

  1. right-click on the scatterplot of interest
  2. choose 'Set Marker Size'
  3. Set 'Size by' to 'ACS' (scroll down list of columns to see it)
This is inelegant, but is the best way I have come up with so far!
